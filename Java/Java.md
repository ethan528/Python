# Java

### 클래스와 객체

클래스로 객체 정의 및 생성

객체 = 속성(변수) + 기능(메서드)

객체 = 모든 인스턴스

인스턴스 = 특정 클래스로 생성된 객체

인스턴스화 = 클래스 > 인스턴스(객체)

소스파일명은 public class명과 일치해야 함

하나의 소스파일엔 하나의 public class만 존재

### 객체 생성

클래스명 참조변수명 = new 클래스명();

변수.변수 = ;

변수.메서드();

다수의 참조변수로 하나의 인스턴스 참조 가능

참조변수 = 참조변수;
